# 🚀 Cosmic Pioneers - Stardance

This project was built for the **Hack Club Stardance** event! It is an interactive web platform designed to explore and celebrate the life stories, milestones, and incredible achievements of 6 pioneering female astronauts who changed the course of space history.

## 🌟 How It Works
When you enter the website, you are greeted with a beautiful space-themed home page showcasing the portraits of the astronauts. Clicking on any astronaut's photo takes you to a **custom-themed profile page** designed around her personality, achievements, and the things she loved!

## 👩‍🚀 Featured Pioneers
* **Valentina Tereshkova** - The first woman in space.
* **Svetlana Savitskaya** - The first woman to perform a spacewalk.
* **Sally Ride** - The first American woman in space.
* **Mae Jemison** - The first African-American woman in space.
* **Peggy Whitson** - Record-breaking NASA astronaut with the most total time in space.
* **Christina Koch** - Leader of the first all-female spacewalk and long-duration spaceflight.

## 🛠️ Built With
* **HTML5** - For semantic page structure and multi-page routing.
* **CSS3** - For custom layouts, responsive grids, and personalized themes for each astronaut.
* **JavaScript (JS)** - For adding interactive animations, dynamic page transitions, and handling click events.
* **GitHub Desktop & VS Code** - For version control and development.
