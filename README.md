
Cosmic Pioneers is a web project I built to showcase the incredible journeys and profiles of 6 pioneering astronauts who have shaped space history.
I built the entire website using HTML and CSS for a clean and structured layout. I designed and organized the astronaut profile grids and navigation sections by hand. No generative AI tools or LLMs were used to write the code or content for this project.
Features
- Profiles of 6 legendary pioneering astronauts.
- Clean, structured multi-section layout.
- Responsive design tailored for web viewing.
